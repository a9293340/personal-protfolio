測試滑鼠滾輪縮放、水平拖曳、視覺特效增強

⌨️ 鍵盤快捷鍵

Ctrl + 滾輪 縮放時間軸
Ctrl + +/- 放大/縮小
Ctrl + 0 重置縮放
← → 左右滾動
滑鼠拖曳 拖動時間軸

🧪 測試步驟

  • 縮放測試: 按住 Ctrl/Cmd + 滾輪縮放時間軸
  • 拖曳測試: 滑鼠點擊並拖曳時間軸左右移動
  • 懸停效果: 滑鼠懸停在節點上查看增強動畫
  • 鍵盤操作: 使用方向鍵和快捷鍵控制
  • 路徑動畫: 觀察時間軸路徑的流動光效

  • 流暢的拖曳體驗
  • 節點懸停時脈動發光效果
  • 路徑上有流動的光線
